## Bild

### Mit Markdown

Die [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/#images) ermöglicht es Ihnen, Bilder mit dem folgenden Code hinzuzufügen

```md
![title](/path/image.jpg)
```

Beachten Sie, dass die Bilddateigröße weniger als 5 MB betragen muss. Andernfalls empfehlen wir das Hosting bei einem Dienst wie [Cloudinary](https://cloudinary.com/) oder [S3](https://aws.amazon.com/s3/). Diese URL können Sie dann verwenden und einbetten.

### Mit Einbettungen

Für mehr Anpassungsmöglichkeiten bei Bildern können Sie auch [Einbettungen](/writing-content/embed) verwenden, um Bilder hinzuzufügen

```html
<img height="200" src="/path/image.jpg" />
```

</Tip>

### iFrames

Lädt eine andere HTML-Seite innerhalb des Dokuments. Am häufigsten wird dies zum Einbetten von Videos verwendet.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
