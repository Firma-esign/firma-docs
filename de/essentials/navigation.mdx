Das Navigationsmenü ist die Liste der Links auf jeder Website.

Sie werden `docs.json` wahrscheinlich jedes Mal aktualisieren, wenn Sie eine neue Seite hinzufügen. Seiten erscheinen nicht automatisch.

## Navigations-Syntax

Unsere Navigations-Syntax ist rekursiv, was bedeutet, dass Sie verschachtelte Navigationsgruppen erstellen können. Sie müssen `.mdx` nicht in Seitennamen einschließen.

<CodeGroup>

```json Regular Navigation
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": ["quickstart"]
        }
      ]
    }
  ]
}
```

```json Nested Navigation
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": [
            "quickstart",
            {
              "group": "Nested Reference Pages",
              "pages": ["nested-reference-page"]
            }
          ]
        }
      ]
    }
  ]
}
```

</CodeGroup>

## Ordner

Legen Sie Ihre MDX-Dateien einfach in Ordner ab und aktualisieren Sie die Pfade in `docs.json`.

Um beispielsweise eine Seite unter `https://yoursite.com/your-folder/your-page` zu haben, würden Sie einen Ordner namens `your-folder` erstellen, der eine MDX-Datei namens `your-page.mdx` enthält.

<Warning>

Sie können `api` nicht als Namen eines Ordners verwenden, es sei denn, Sie verschachteln ihn in einem anderen Ordner. Mintlify verwendet Next.js, das den obersten Ordner `api` für interne Serveraufrufe reserviert. Ein Ordnername wie `api-reference` wäre akzeptabel.

</Warning>

```json Navigation With Folder
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

## Versteckte Seiten

MDX-Dateien, die nicht in `docs.json` enthalten sind, erscheinen nicht in der Seitenleiste, sind aber über die Suchleiste und durch direkte Verlinkung zugänglich.
