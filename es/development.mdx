<Info>
  **Requisitos previos**:
  - Node.js versión 19 o superior
  - Un repositorio de documentación con un archivo `docs.json`
</Info>

Sigue estos pasos para instalar y ejecutar Mintlify en tu sistema operativo.

<Steps>
<Step title="Instala la CLI de Mintlify">

```bash
npm i -g mint
```
</Step>

<Step title="Previsualiza localmente">

Navega al directorio de documentación donde se encuentra tu archivo `docs.json` y ejecuta el siguiente comando:

```bash
mint dev
```

Estará disponible una vista previa local de tu documentación en `http://localhost:3000`.

</Step>
</Steps>

## Puertos personalizados

De forma predeterminada, Mintlify utiliza el puerto 3000. Puedes personalizar el puerto en el que se ejecuta Mintlify utilizando la opción `--port`. Por ejemplo, para ejecutar Mintlify en el puerto 3333, usa este comando:

```bash
mint dev --port 3333
```

Si intentas ejecutar Mintlify en un puerto que ya está en uso, utilizará el siguiente puerto disponible:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Versiones de Mintlify

Ten en cuenta que cada versión de la CLI está asociada a una versión específica de Mintlify. Si tu vista previa local no coincide con la versión de producción, actualiza la CLI:

```bash
npm mint update
```

## Validación de enlaces

La CLI puede ayudarte a validar los enlaces de tu documentación. Para identificar cualquier enlace roto, usa el siguiente comando:

```bash
mint broken-links
```

## Formato de código

Sugerimos utilizar extensiones en tu IDE para reconocer y formatear MDX. Si usas VSCode, considera la [extensión MDX para VSCode](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) para el resaltado de sintaxis y [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) para el formateo de código.

## Resolución de problemas

<AccordionGroup>
  <Accordion title='Error: Could not load the "sharp" module using the darwin-arm64 runtime'>

    Esto puede deberse a una versión desactualizada de Node. Prueba lo siguiente:
    1. Elimina la versión de la CLI instalada actualmente: `npm remove -g mint`
    2. Actualiza a Node v19 o superior.
    3. Reinstala la CLI: `npm i -g mint`
  </Accordion>

  <Accordion title="Problema: se produce un error desconocido">
  
    Solución: ve a la raíz de tu dispositivo y elimina la carpeta `~/.mintlify`. A continuación, ejecuta `mint dev` de nuevo.
  </Accordion>
</AccordionGroup>

¿Tienes curiosidad por saber qué ha cambiado en la última versión de la CLI? Consulta el [registro de cambios de la CLI](https://www.npmjs.com/package/mintlify?activeTab=versions).
