## Código en línea

Para marcar una `palabra` o una `frase` como código, enciérrala entre acentos graves (`).

```
Para marcar una `palabra` o una `frase` como código, enciérrala entre acentos graves (`).
```

## Bloques de código

Utiliza [bloques de código delimitados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) encerrando el código entre tres acentos graves y añadiendo, tras los acentos iniciales, el lenguaje de programación de tu fragmento para obtener resaltado de sintaxis. Opcionalmente, también puedes escribir el nombre de tu código después del lenguaje de programación.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
