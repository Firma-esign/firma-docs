## Imagen

### Usando Markdown

La [sintaxis de markdown](https://www.markdownguide.org/basic-syntax/#images) te permite añadir imágenes con el siguiente código

```md
![título](/ruta/imagen.jpg)
```

Ten en cuenta que el tamaño del archivo de imagen debe ser inferior a 5 MB. De lo contrario, recomendamos alojarlas en un servicio como [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/). A continuación, puedes usar esa URL e insertarla.

### Usando inserciones

Para obtener más personalización con las imágenes, también puedes utilizar [inserciones](/writing-content/embed) para añadirlas

```html
<img height="200" src="/ruta/imagen.jpg" />
```

Mintlify admite [etiquetas HTML en Markdown](https://www.markdownguide.org/basic-syntax/#html). Esto es útil si prefieres las etiquetas HTML a la sintaxis de Markdown y te permite crear documentación con flexibilidad infinita.

</Tip>

### iFrames

Carga otra página HTML dentro del documento. Se utiliza normalmente para insertar vídeos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
