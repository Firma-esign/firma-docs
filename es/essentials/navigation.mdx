El menú de navegación es la lista de enlaces presente en cada sitio web.

Es probable que actualices `docs.json` cada vez que añadas una nueva página. Las páginas no aparecen automáticamente.

## Sintaxis de navegación

Nuestra sintaxis de navegación es recursiva, lo que significa que puedes crear grupos de navegación anidados. No necesitas incluir `.mdx` en los nombres de página.

<CodeGroup>

```json Regular Navigation
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": ["quickstart"]
        }
      ]
    }
  ]
}
```

```json Nested Navigation
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": [
            "quickstart",
            {
              "group": "Nested Reference Pages",
              "pages": ["nested-reference-page"]
            }
          ]
        }
      ]
    }
  ]
}
```

</CodeGroup>

## Carpetas

Simplemente coloca tus archivos MDX en carpetas y actualiza las rutas en `docs.json`.

Por ejemplo, para tener una página en `https://tusitio.com/tu-carpeta/tu-pagina`, crearías una carpeta llamada `tu-carpeta` que contenga un archivo MDX llamado `tu-pagina.mdx`.

<Warning>

No puedes usar `api` como nombre de una carpeta a menos que la anides dentro de otra carpeta. Mintlify utiliza Next.js, que reserva la carpeta `api` de nivel superior para llamadas internas al servidor. Se aceptaría un nombre de carpeta como `api-reference`.

</Warning>

```json Navigation With Folder
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

## Páginas ocultas

Los archivos MDX que no se incluyan en `docs.json` no aparecerán en la barra lateral, pero son accesibles a través de la barra de búsqueda y enlazándolos directamente.
