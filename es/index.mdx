## ¿Qué es Firma?

Firma ofrece flujos de firma alojados y un editor de plantillas incrustable para que puedas agregar la firma de documentos a tu producto con un esfuerzo de desarrollo mínimo. Puedes:

- Crear espacios de trabajo aislados y seguros para cada uno de tus clientes, de modo que sus datos permanezcan separados.
- Incrustar un editor de plantillas para que los usuarios puedan diseñar documentos desde dentro de tu app.
- Incrustar la experiencia de firma para que los firmantes puedan firmar sin salir de tu interfaz.
- Usar flujos de trabajo para habilitar funcionalidad nativa dentro de tu app, de modo que tengas una experiencia de firma electrónica totalmente white-label.
