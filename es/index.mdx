## ¿Qué es Firma?

Firma ofrece flujos de firma alojados y un editor de plantillas integrable para que puedas añadir firma de documentos a tu producto con un esfuerzo mínimo de desarrollo. Puedes:

- Crear espacios de trabajo aislados y seguros para cada uno de tus clientes, de modo que sus datos permanezcan separados.
- Integrar un editor de plantillas para que los usuarios puedan diseñar documentos desde dentro de tu aplicación.
- Integrar la experiencia de firma para que los firmantes puedan firmar sin salir de tu interfaz.
- Utilizar flujos de trabajo para habilitar funcionalidad nativa dentro de tu aplicación, de forma que tengas una experiencia de firma electrónica totalmente con marca blanca.
