## Code en ligne

Pour indiquer qu'un `mot` ou une `expression` est du code, entourez-le d'accents graves (`).


## Blocs de code

Utilisez les [blocs de code délimités](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) en entourant votre code de trois accents graves, et faites suivre les accents graves d'ouverture du langage de programmation de votre extrait pour obtenir la coloration syntaxique. Vous pouvez également, en option, écrire le nom de votre code après le langage de programmation.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
