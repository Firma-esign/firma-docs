<Info>
  **Prerequisiti**:
  - Node.js versione 19 o superiore
  - Un repository di documentazione con un file `docs.json`
</Info>

Segui questi passaggi per installare ed eseguire Mintlify sul tuo sistema operativo.

<Steps>
<Step title="Installa la CLI di Mintlify">

```bash
npm i -g mint
```
</Step>

<Step title="Anteprima locale">

Vai nella directory della tua documentazione dove si trova il file `docs.json` ed esegui il seguente comando:

```bash
mint dev
```

Sarà disponibile un'anteprima locale della tua documentazione all'indirizzo `http://localhost:3000`.

</Step>
</Steps>

## Porte personalizzate

Per impostazione predefinita, Mintlify utilizza la porta 3000. Puoi personalizzare la porta su cui gira Mintlify usando il flag `--port`. Ad esempio, per eseguire Mintlify sulla porta 3333, usa questo comando:

```bash
mint dev --port 3333
```

Se provi a eseguire Mintlify su una porta già in uso, verrà utilizzata la successiva porta disponibile:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Versioni di Mintlify

Nota che ogni release della CLI è associata a una versione specifica di Mintlify. Se l'anteprima locale non è allineata con la versione di produzione, aggiorna la CLI:

```bash
npm mint update
```

## Validazione dei link

La CLI può aiutarti a validare i link della tua documentazione. Per identificare eventuali link non funzionanti, usa il seguente comando:

```bash
mint broken-links
```

## Formattazione del codice

Ti consigliamo di utilizzare estensioni per il tuo IDE che riconoscano e formattino MDX. Se usi VSCode, prendi in considerazione l'[estensione MDX per VSCode](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) per l'evidenziazione della sintassi e [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) per la formattazione del codice.

## Risoluzione dei problemi

<AccordionGroup>
  <Accordion title='Errore: Could not load the "sharp" module using the darwin-arm64 runtime'>

    Potrebbe essere dovuto a una versione obsoleta di node. Prova quanto segue:
    1. Rimuovi la versione della CLI attualmente installata: `npm remove -g mint`
    2. Aggiorna a Node v19 o superiore.
    3. Reinstalla la CLI: `npm i -g mint`
  </Accordion>

  <Accordion title="Problema: si verifica un errore sconosciuto">
  
    Soluzione: vai nella root del tuo dispositivo ed elimina la cartella `~/.mintlify`. Quindi esegui di nuovo `mint dev`.
  </Accordion>
</AccordionGroup>

Vuoi sapere cosa è cambiato nell'ultima versione della CLI? Consulta il [changelog della CLI](https://www.npmjs.com/package/mintlify?activeTab=versions).
