## Codice inline

Per indicare una `parola` o una `frase` come codice, racchiudila tra backtick (`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

## Blocchi di codice

Usa i [blocchi di codice fenced](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) racchiudendo il codice tra tre backtick e facendo seguire ai backtick iniziali il linguaggio di programmazione dello snippet per ottenere l'evidenziazione della sintassi. Facoltativamente, puoi anche scrivere il nome del codice dopo il linguaggio di programmazione.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
