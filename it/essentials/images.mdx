## Immagine

### Usando Markdown

La [sintassi markdown](https://www.markdownguide.org/basic-syntax/#images) ti consente di aggiungere immagini usando il seguente codice

```md
![title](/path/image.jpg)
```

Nota che la dimensione del file immagine deve essere inferiore a 5MB. Altrimenti, ti consigliamo l'hosting su un servizio come [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/). Puoi quindi usare quell'URL e incorporarlo.

### Usando i contenuti incorporati

Per ottenere maggiore personalizzazione con le immagini, puoi anche usare i [contenuti incorporati](/writing-content/embed) per aggiungere immagini

```html
<img height="200" src="/path/image.jpg" />
```

### iFrame

Carica un'altra pagina HTML all'interno del documento. Usato più comunemente per incorporare video.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
