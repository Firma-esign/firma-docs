## Obraz

### Użycie Markdown

[Składnia markdown](https://www.markdownguide.org/basic-syntax/#images) umożliwia dodawanie obrazów za pomocą następującego kodu

```md
![title](/path/image.jpg)
```

Pamiętaj, że rozmiar pliku obrazu musi być mniejszy niż 5 MB. W przeciwnym razie zalecamy hosting w usłudze takiej jak [Cloudinary](https://cloudinary.com/) lub [S3](https://aws.amazon.com/s3/). Możesz następnie użyć tego adresu URL i osadzić obraz.

### Użycie osadzeń

Aby uzyskać większe możliwości dostosowania obrazów, możesz także używać [osadzeń](/writing-content/embed) do dodawania obrazów

```html
<img height="200" src="/path/image.jpg" />
```

## Osadzenia i elementy HTML

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
  style={{ width: '100%', borderRadius: '0.5rem' }}
></iframe>

<br />

<Tip>

Mintlify obsługuje [znaczniki HTML w Markdown](https://www.markdownguide.org/basic-syntax/#html). Jest to pomocne, jeśli wolisz znaczniki HTML od składni Markdown i pozwala tworzyć dokumentację z nieograniczoną elastycznością.

</Tip>

### iFrames

Ładują inną stronę HTML wewnątrz dokumentu. Najczęściej używane do osadzania filmów.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
