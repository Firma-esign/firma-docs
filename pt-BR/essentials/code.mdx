## Código inline

Para marcar uma `palavra` ou `frase` como código, envolva-a em crases (`).

```
Para marcar uma `palavra` ou `frase` como código, envolva-a em crases (`).
```

## Blocos de código

Use [blocos de código cercados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) envolvendo o código em três crases e siga as crases iniciais com a linguagem de programação do seu snippet para obter realce de sintaxe. Opcionalmente, você também pode escrever o nome do seu código após a linguagem de programação.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
