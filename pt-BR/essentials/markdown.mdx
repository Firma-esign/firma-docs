## Títulos

Melhor usados para cabeçalhos de seção.

```md
## Títulos
```

### Subtítulos

Melhor usados para cabeçalhos de subseção.

```md
### Subtítulos
```

<Tip>

Cada **título** e **subtítulo** cria uma âncora e também aparece no sumário à direita.

</Tip>

## Formatação de texto

Suportamos a maior parte da formatação markdown. Basta adicionar `**`, `_` ou `~` ao redor do texto para formatá-lo.

| Estilo        | Como escrever     | Resultado       |
| ------------- | ----------------- | --------------- |
| Negrito       | `**negrito**`     | **negrito**     |
| Itálico       | `_itálico_`       | _itálico_       |
| Tachado       | `~tachado~`       | ~tachado~       |

Você pode combiná-los. Por exemplo, escreva `**_negrito e itálico_**` para obter texto **_negrito e itálico_**.

Você precisa usar HTML para escrever texto sobrescrito e subscrito. Ou seja, adicione `<sup>` ou `<sub>` ao redor do seu texto.

| Tamanho do texto | Como escrever            | Resultado              |
| ---------------- | ------------------------ | ---------------------- |
| Sobrescrito      | `<sup>sobrescrito</sup>` | <sup>sobrescrito</sup> |
| Subscrito        | `<sub>subscrito</sub>`   | <sub>subscrito</sub>   |

## Vinculando a páginas

Você pode adicionar um link envolvendo o texto em `[]()`. Você escreveria `[link para o google](https://google.com)` para [link para o google](https://google.com).

Links para páginas nos seus docs precisam ser relativos à raiz. Basicamente, você deve incluir o caminho de pasta completo. Por exemplo, `[link para texto](/writing-content/text)` vincula à página "Text" na nossa seção de componentes.

Links relativos como `[link para texto](../text)` abrirão mais lentamente porque não conseguimos otimizá-los com a mesma facilidade.

## Blockquotes

### Linha única

Para criar uma citação em bloco, adicione um `>` na frente de um parágrafo.

> Dorothy a seguiu através de muitos dos belos aposentos em seu castelo.

```md
> Dorothy a seguiu através de muitos dos belos aposentos em seu castelo.
```

### Múltiplas linhas

> Dorothy a seguiu através de muitos dos belos aposentos em seu castelo.
>
> A Bruxa mandou que ela limpasse as panelas e chaleiras, varresse o chão e mantivesse o fogo alimentado com lenha.

```md
> Dorothy a seguiu através de muitos dos belos aposentos em seu castelo.
>
> A Bruxa mandou que ela limpasse as panelas e chaleiras, varresse o chão e mantivesse o fogo alimentado com lenha.
```

### LaTeX

O Mintlify suporta [LaTeX](https://www.latex-project.org) através do componente Latex.

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
