<Info>
  **Pré-requisitos**:
  - Node.js versão 19 ou superior
  - Um repositório de documentação com um ficheiro `docs.json`
</Info>

Siga estes passos para instalar e executar o Mintlify no seu sistema operativo.

<Steps>
<Step title="Instalar o CLI do Mintlify">

```bash
npm i -g mint
```
</Step>

<Step title="Pré-visualizar localmente">

Navegue até ao diretório da sua documentação onde se encontra o ficheiro `docs.json` e execute o seguinte comando:

```bash
mint dev
```

Uma pré-visualização local da sua documentação ficará disponível em `http://localhost:3000`.

</Step>
</Steps>

## Portas personalizadas

Por defeito, o Mintlify utiliza a porta 3000. Pode personalizar a porta em que o Mintlify é executado utilizando a flag `--port`. Por exemplo, para executar o Mintlify na porta 3333, utilize este comando:

```bash
mint dev --port 3333
```

Se tentar executar o Mintlify numa porta que já esteja em uso, será utilizada a próxima porta disponível:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Versões do Mintlify

Tenha em atenção que cada versão do CLI está associada a uma versão específica do Mintlify. Se a sua pré-visualização local não estiver alinhada com a versão em produção, atualize o CLI:

```bash
npm mint update
```

## Validar ligações

O CLI pode ajudar a validar ligações na sua documentação. Para identificar quaisquer ligações inválidas, utilize o seguinte comando:

```bash
mint broken-links
```

## Formatação de código

Sugerimos que utilize extensões no seu IDE para reconhecer e formatar MDX. Se for utilizador de VSCode, considere a [extensão MDX do VSCode](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) para realce de sintaxe e o [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) para formatação de código.

## Resolução de problemas

<AccordionGroup>
  <Accordion title='Erro: não foi possível carregar o módulo "sharp" utilizando o runtime darwin-arm64'>

    Isto pode dever-se a uma versão desatualizada do node. Tente o seguinte:
    1. Remova a versão atualmente instalada do CLI: `npm remove -g mint`
    2. Atualize para o Node v19 ou superior.
    3. Reinstale o CLI: `npm i -g mint`
  </Accordion>

  <Accordion title="Problema: aparece um erro desconhecido">
  
    Solução: aceda à raiz do seu dispositivo e elimine a pasta `~/.mintlify`. Em seguida, execute `mint dev` novamente.
  </Accordion>
</AccordionGroup>

Curioso sobre o que mudou na última versão do CLI? Consulte o [changelog do CLI](https://www.npmjs.com/package/mintlify?activeTab=versions).
