## Código inline

Para assinalar uma `palavra` ou `frase` como código, envolva-a em acentos graves (`).

```
Para assinalar uma `palavra` ou `frase` como código, envolva-a em acentos graves (`).
```

## Blocos de código

Utilize [blocos de código delimitados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) envolvendo o código em três acentos graves e adicione, a seguir aos acentos iniciais, a linguagem de programação do seu excerto para obter realce de sintaxe. Opcionalmente, pode também escrever o nome do código depois da linguagem de programação.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
