## Imagem

### Utilizando Markdown

A [sintaxe markdown](https://www.markdownguide.org/basic-syntax/#images) permite-lhe adicionar imagens utilizando o seguinte código

```md
![title](/path/image.jpg)
```

Tenha em atenção que o tamanho do ficheiro de imagem deve ser inferior a 5 MB. Caso contrário, recomendamos o alojamento num serviço como [Cloudinary](https://cloudinary.com/) ou [S3](https://aws.amazon.com/s3/). Pode depois utilizar esse URL e incorporá-lo.

### Utilizando incorporações

Para obter maior personalização com imagens, também pode utilizar [incorporações](/writing-content/embed) para adicionar imagens

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Carrega outra página HTML dentro do documento. Utilizado mais frequentemente para incorporar vídeos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
