## Títulos

Utilizados idealmente para cabeçalhos de secção.

```md
## Títulos
```

### Subtítulos

Utilizados idealmente para cabeçalhos de subsecção.

```md
### Subtítulos
```

<Tip>

Cada **título** e **subtítulo** cria uma âncora e também aparece no índice à direita.

</Tip>

## Formatação de texto

Suportamos a maior parte da formatação markdown. Basta adicionar `**`, `_` ou `~` em torno do texto para o formatar.

| Estilo         | Como escrever      | Resultado       |
| -------------- | ------------------ | --------------- |
| Negrito        | `**negrito**`      | **negrito**     |
| Itálico        | `_itálico_`        | _itálico_       |
| Rasurado       | `~rasurado~`       | ~rasurado~      |

Pode combiná-los. Por exemplo, escreva `**_negrito e itálico_**` para obter texto **_negrito e itálico_**.

Precisa de utilizar HTML para escrever texto sobrescrito e subscrito. Ou seja, adicione `<sup>` ou `<sub>` em torno do texto.

| Tamanho do texto | Como escrever            | Resultado              |
| ---------------- | ------------------------ | ---------------------- |
| Sobrescrito      | `<sup>sobrescrito</sup>` | <sup>sobrescrito</sup> |
| Subscrito        | `<sub>subscrito</sub>`   | <sub>subscrito</sub>   |

## Ligações para páginas

Pode adicionar uma ligação envolvendo o texto em `[]()`. Escreveria `[ligação para o google](https://google.com)` para [ligação para o google](https://google.com).

As ligações para páginas na sua documentação devem ser relativas à raiz. Basicamente, deve incluir o caminho completo da pasta. Por exemplo, `[ligação para texto](/writing-content/text)` liga à página "Texto" na nossa secção de componentes.

Ligações relativas como `[ligação para texto](../text)` abrirão mais lentamente porque não as conseguimos otimizar com a mesma facilidade.

## Citações em bloco

### Linha única

Para criar uma citação em bloco, adicione um `>` antes de um parágrafo.

> A Dorothy seguiu-a através de muitas das belas salas do seu castelo.

```md
> A Dorothy seguiu-a através de muitas das belas salas do seu castelo.
```

### Várias linhas

> A Dorothy seguiu-a através de muitas das belas salas do seu castelo.
>
> A Bruxa ordenou-lhe que limpasse os potes e as chaleiras, varresse o chão e mantivesse o fogo alimentado com lenha.

```md
> A Dorothy seguiu-a através de muitas das belas salas do seu castelo.
>
> A Bruxa ordenou-lhe que limpasse os potes e as chaleiras, varresse o chão e mantivesse o fogo alimentado com lenha.
```

### LaTeX

O Mintlify suporta [LaTeX](https://www.latex-project.org) através do componente Latex.

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
