## Criar um excerto personalizado

**Pré-condição**: tem de criar o seu ficheiro de excerto no diretório `snippets`.

<Note>
  Qualquer página no diretório `snippets` será tratada como um excerto e não
  será apresentada como uma página autónoma. Se pretende criar uma página
  autónoma a partir do excerto, importe-o para outro ficheiro e utilize-o
  como componente.
</Note>

### Exportação por defeito

1. Adicione ao seu ficheiro de excerto o conteúdo que pretende reutilizar em
   várias localizações. Opcionalmente, pode adicionar variáveis que podem ser
   preenchidas através de props quando importa o excerto.

```mdx snippets/my-snippet.mdx
Hello world! This is my content I want to reuse across pages. My keyword of the
day is {word}.
```

<Warning>
  O conteúdo que pretende reutilizar tem de estar dentro do diretório `snippets`
  para que a importação funcione.
</Warning>

2. Importe o excerto para o ficheiro de destino.

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Header

Lorem impsum dolor sit amet.

<MySnippet word="bananas" />
```

### Variáveis reutilizáveis

1. Exporte uma variável a partir do seu ficheiro de excerto:

```mdx snippets/path/to/custom-variables.mdx
export const myName = 'my name';

export const myObject = { fruit: 'strawberries' };
```

2. Importe o excerto a partir do ficheiro de destino e utilize a variável:

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hello, my name is {myName} and I like {myObject.fruit}.
```

### Componentes reutilizáveis

1. Dentro do seu ficheiro de excerto, crie um componente que receba props
   exportando o seu componente sob a forma de uma arrow function.

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... snippet content ...</p>
  </div>
);
```

<Warning>
  O MDX não compila dentro do corpo de uma arrow function. Mantenha-se com
  sintaxe HTML quando possível, ou utilize uma exportação por defeito se
  precisar de usar MDX.
</Warning>

2. Importe o excerto para o ficheiro de destino e passe as props

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Custom title'} />
```
