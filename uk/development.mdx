<Info>
  **Передумови**:
  - Node.js версії 19 або вище
  - Репозиторій документації з файлом `docs.json`
</Info>

Виконайте ці кроки, щоб встановити та запустити Mintlify у вашій операційній системі.

<Steps>
<Step title="Встановіть Mintlify CLI">

```bash
npm i -g mint
```
</Step>

<Step title="Попередній перегляд локально">

Перейдіть до каталогу документації, де знаходиться ваш файл `docs.json`, та виконайте наступну команду:

```bash
mint dev
```

Локальний попередній перегляд вашої документації буде доступний за адресою `http://localhost:3000`.

</Step>
</Steps>

## Користувацькі порти

За замовчуванням Mintlify використовує порт 3000. Ви можете налаштувати порт, на якому працює Mintlify, за допомогою прапорця `--port`. Наприклад, щоб запустити Mintlify на порті 3333, використайте цю команду:

```bash
mint dev --port 3333
```

Якщо ви спробуєте запустити Mintlify на порті, який уже використовується, він використає наступний доступний порт:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Версії Mintlify

Зверніть увагу, що кожен випуск CLI пов’язаний з конкретною версією Mintlify. Якщо ваш локальний попередній перегляд не відповідає версії у продакшені, будь ласка, оновіть CLI:

```bash
npm mint update
```

## Валідація посилань

CLI може допомогти з валідацією посилань у вашій документації. Щоб виявити будь-які непрацюючі посилання, використайте наступну команду:

```bash
mint broken-links
```

## Форматування коду

Ми рекомендуємо використовувати розширення для вашої IDE, щоб розпізнавати та форматувати MDX. Якщо ви користувач VSCode, розгляньте [розширення MDX VSCode](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) для підсвічування синтаксису та [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) для форматування коду.

## Усунення несправностей

<AccordionGroup>
  <Accordion title='Помилка: Не вдалося завантажити модуль "sharp" за допомогою середовища виконання darwin-arm64'>

    Це може бути пов’язано із застарілою версією node. Спробуйте наступне:
    1. Видаліть встановлену версію CLI: `npm remove -g mint`
    2. Оновіть Node до версії 19 або вище.
    3. Перевстановіть CLI: `npm i -g mint`
  </Accordion>

  <Accordion title="Проблема: Виникає невідома помилка">
  
    Рішення: Перейдіть до кореневого каталогу вашого пристрою та видаліть теку `~/.mintlify`. Потім знову запустіть `mint dev`.
  </Accordion>
</AccordionGroup>

Цікаво, що змінилося в останній версії CLI? Перегляньте [журнал змін CLI](https://www.npmjs.com/package/mintlify?activeTab=versions).
