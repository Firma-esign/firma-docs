## Вбудований код

Щоб позначити `слово` або `фразу` як код, візьміть його у зворотні лапки (`).

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

## Блоки коду

Використовуйте [огороджені блоки коду](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks), обгортаючи код у три зворотні лапки, а після початкових лапок вкажіть мову програмування вашого сніпета, щоб отримати підсвічування синтаксису. За бажанням ви також можете написати назву вашого коду після мови програмування.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
