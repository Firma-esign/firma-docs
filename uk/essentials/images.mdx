## Зображення

### Використання Markdown

[Синтаксис Markdown](https://www.markdownguide.org/basic-syntax/#images) дозволяє додавати зображення за допомогою наступного коду

```md
![title](/path/image.jpg)
```

Зверніть увагу, що розмір файлу зображення має бути менше 5 МБ. В іншому випадку ми рекомендуємо розміщувати їх на сервісі на кшталт [Cloudinary](https://cloudinary.com/) або [S3](https://aws.amazon.com/s3/). Потім ви можете використати цю URL-адресу та вбудувати.

### Використання вбудовувань

Щоб отримати більше можливостей налаштування зображень, ви також можете використовувати [вбудовування](/writing-content/embed) для додавання зображень

```html
<img height="200" src="/path/image.jpg" />
```

## Вбудовування та HTML-елементи

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
  style={{ width: '100%', borderRadius: '0.5rem' }}
></iframe>

<br />

<Tip>

Mintlify підтримує [HTML-теги у Markdown](https://www.markdownguide.org/basic-syntax/#html). Це корисно, якщо ви віддаєте перевагу HTML-тегам перед синтаксисом Markdown, і дозволяє створювати документацію з необмеженою гнучкістю.

</Tip>

### iFrames

Завантажує іншу HTML-сторінку всередині документа. Найчастіше використовується для вбудовування відео.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
